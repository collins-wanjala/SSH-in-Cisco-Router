# SSH-in-Cisco-Router-in-Cisco-Packet-Tracer

Learn how to configure SSH on a Cisco Router using Cisco Packet Tracer in this step-by-step CCNA lab. In this tutorial, we demonstrate secure remote access, SSH configuration commands, and verification on Cisco devices. This networking lab is perfect for CCNA students, beginners, and network engineers learning Cisco router configuration.

STEPS TO CONFIGURE SSH



1. Cable and assign IP Addresses
2. Configure Ip domain name
3. username and password for local authentication
4. Generate RSA keys 
5. Transport input ssh
6. Test connectivity
